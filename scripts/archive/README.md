# Archived Scripts

This directory contains helper scripts that were used during development but are no longer needed for production.

## Scripts

- `check_session_setup.py` - Session debugging script
- `create_active_games_table.py` - DynamoDB table creation helper
- `create_dynamo_table.py` - DynamoDB setup helper
- `decode_session.py` - Session debugging tool
- `reproduce_token_issue.py` - Token debugging script
- `setup_auth.py` - Auth setup helper
- `verify_dynamo_connection.py` - DynamoDB connection test

## Note

These scripts are kept for reference but are not part of the production deployment.
